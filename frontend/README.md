# Frontend (React)

Placeholder React + Vite application for the future ADHD Board UI.

## Quick start

```bash
cd frontend
cp .env.example .env  # optional: configure VITE_API_URL
npm install
npm run dev
```

By default the app points to `http://localhost:5000`. Override via `VITE_API_URL`
in `frontend/.env` or by passing the Docker build arg (see Dockerfile).
